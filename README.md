# Expense-Tracker

A simple Java Swing-based Expense Tracker application that helps users manage daily expenses efficiently.
This project allows users to add expenses, view them in a table, and calculate the total expense amount.

**Features:**
* Add expense details:
*  Date
*  Category
*  Amount
* Display expenses in a table format
* Calculate total expenses instantly
* Store expense records in a text file (expenses.txt)
* User-friendly GUI using Java Swing
* Error handling for invalid inputs

**Technologies Used**
* Java
* Java Swing (GUI)
* File Handling
* ArrayList
* JTable & DefaultTableModel
**Output**


**Main Interface**
* Input fields for date, category, and amount
* Add Expense button
* Total Expense button
* Expense records displayed in a table

**How It Works**
  
1 User enters:
*  Date
*  Expense Category
*  Amount

2 Click Add Expense
*  Expense gets added to the table
*  Data is stored in expenses.txt
  
3 Click Total
*  Displays the total expenses using a popup message
